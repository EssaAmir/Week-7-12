## Project Overview
This project upgrades the authentication system from text files to a secure **SQLite Database**. It includes:
* **Data Migration:** Moving users from `users.txt` to the database.
* **Data Pipeline:** Loading CSV data (Incidents, Tickets, Metadata) using Pandas.
* **CRUD Operations:** Functions to Create, Read, Update, and Delete security incidents.

## How to Run
1.  Install dependencies: `pip install -r requirements.txt`
2.  Run the script: `python week_8_lab.py`
